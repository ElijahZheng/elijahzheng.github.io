Git入门

Author: Elijah Zheng Date: July 7, 2017  10:36:00 Category: 爬坑

一、git实现简单的文件上传

（这里会用到coding）->

创建自己的项目

  • 基础选项
    • 项目名称 – 建议用全英文
    • 属性 – 公开
  • 初始化仓库
    • 启用README.md文件初始化项目

clone到本地仓库


选择要clone到的文件夹，比如我要clone到桌面，我的桌面名字为Desktop

在clone下来的文件夹里新建文件

  • 然后进入clone好的文件夹里新建一个自定义文件，比如新建一个666.txt

查看此时本地仓库的状态

将修改文件添加到暂缓区（index）

确认无误后添加到本地仓库（repository）

附：(没有则跳过)
如果提示出现please tell me who you are，按照求输入就行了.

本地修改完成后提交到远程仓库（remote）

二、修改工作区文件，传到远程仓库

  • 修改666.txt的内容,添加信息 hello world

  • 添加到暂缓区
    $ git add 666.txt

  • 添加到本地仓库
    $ git commit -m '我添加了一些信息'

  • 上传到远程仓库
    $ git push

然后就可以看到上传的文件和备注的一些信息了。
